Builder is the ultimate page layout builder for WordPress.

== Description ==

Builder allows you to create unique and high quality page layouts in WordPress.

The plugin uses best practices to deliver great looking pages to your users. Our main focus with this plugin is the ability to create very high quality designs and to provide the best integration with themes and plugins.

> <strong>Use Builder in your premium themes</strong><br>
> Builder is an open source page builder. You can use it to build any free or premium theme. It's easy to ship your designs with Builder and create new additional modules.

= Features =
* Frontend live editing.
* Template library
* Custom widgets
* Integration with inbuilt WordPress widgets
* Custom page template included
* Extension manager for theme and new widget integration
* Revision control
* Section dividers
* Content animations
* Custom colors
* Custom page fonts

= Well Integrated With WordPress =
Builder works with any standard compliant WordPress theme. It's fully compatible with other plugins like Yoast WP SEO or contact form builders.

= Export/Import Sections, Layouts or whole Pages =
Export any page section or the whole page. Reuse pages, sections or widgets you created.

= Mobile Responsive =
It's easy to create responsive websites in Builder. You can customize responsive layouts on element to make it looks perfect on all mobile devices, ensuring your site is mobile-ready.

== Installation ==

1. Upload the `builder` folder to the `/wp-content/plugins/` directory
2. Activate the Builder plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Builder` menu that appears in your admin menu

== Frequently Asked Questions ==

= Which themes does this plugin support? =
Builder works with all standard compliant WordPress themes. All the plugin needs is the content area of a page to be present for it to work.
