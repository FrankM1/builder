Builder is the ultimate page layout builder for WordPress.

== Description ==

Builder allows you to create unique and high quality page layouts in WordPress. The plugin uses best practices to deliver great looking pages to your users.

= Features =

* Frontend live editing.
* Template library
* Custom widgets
* Integration with inbuilt WordPress widgets
* Custom page template included
* Extension manager for theme and new widget integration

== Installation ==

1. Upload the `builder` folder to the `/wp-content/plugins/` directory
2. Activate the Builder plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Builder` menu that appears in your admin menu

== Frequently Asked Questions ==

= Which themes does this plugin support? =
Builder works with all themes. All the plugin needs it the content area of a page to be present for it to work.
